# Hackintosh_2600X_B450I_EFI
Clover EFI folder for Ryzen Hackintosh
Works well with macOS Catalina 10.15.1

 - Processor:Ryzen 5 2600X (OC to 3.8GHz)
 - RAM:Kingston HyperX predator 8GB DDR4 3200*2 (OC to 3400MHz)
 - Motherboard:ASUS ROG STRIX B450I
 - Wi-Fi&Bluetooth:BCM94360CS2 11ac 2T2R (using a M2 A-E Key adapter to replace the original RTL8822BE on the mother board)
 - SSD:HIKVision C2000 1TB(NVMe)
 - HDD:Seagate 2TB
 - Graphics: Dataland RX580 4GB
